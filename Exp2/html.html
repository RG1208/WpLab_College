HTML

What is HTML?

HTML, or HyperText Markup Language, is the standard language used to create and design webpages. It forms the backbone of web content by structuring and presenting text, images, links, and other elements on the internet.

Basic Structure

HTML uses "tags" to define elements. Tags are enclosed in angle brackets.

An HTML document is a text file with an .html or .htm extension.

Common attributes

id: A unique identifier for an element.
class: A class name for styling with CSS.
style: Inline CSS styles.
alt: Alternative text for images.
